## New discrepancies not previously documented in the source file

1. **DM-008 — Duplicate "Total Hours (Today)" KPI label collision.** When Period = "Today", both the always-today card and the period-following card render the identical heading text "Total Hours (Today)" while showing different values (0h vs 12/12 billable) with no visual distinction — confusing/misleading, and not covered by the existing "Total Hours (Period)" relabeling logic documented for "This month".

2. **DM-011/DM-023/DM-026 — Custom date-range queries always silently include "today" (major).** Root-caused via network trace: any custom range picked via the date-range picker that does not already span today gets clamped as `from = min(selectedFrom, today)`, `to = max(selectedTo, today)` for the `/timelogs` endpoint that feeds KPI cards, Project Distribution, Category Split, and the Weekly Progress Chart. This means genuinely empty periods (fully future or fully past, excluding today) never show the correct "0h"/"No time logged in this period" empty state — they show stale, wrong, non-zero data instead. Confirmed reproducibly twice (future range Aug 15–20, past range Jan 5–10) with full request/response evidence. Notably, the separately-implemented Team Activities widget (its own `/workspaces/.../team-activities` endpoint) does NOT have this bug and correctly showed the empty state for the exact same picked ranges — producing visibly contradictory data side-by-side on the same dashboard screen. This is the most significant new finding this session and should be filed as a new bug.

3. **DM-031 — Dashboard's own widgets don't auto-refresh after Quick Timer Stop.** The timer stop mechanism itself works correctly and data is saved server-side without loss (confirmed via `/timer/stop` response body), but the dashboard's "Today's Activity Feed" and "Total Hours (Today)" KPI card remain stale for at least 6+ seconds after Stop and only update on a full manual page reload. This is separate from the already-documented Note-dialog-timing gap (#656).
