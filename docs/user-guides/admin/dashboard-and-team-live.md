# Admin: Dashboard and team live

## Dashboard

1. Go to **Dashboard** (`/dashboard`).
2. Choose a date range (e.g. last 7, 30, or 90 days).
3. Review:
   - Total hours and billable amounts
   - Breakdown by project and by member
   - Charts for trends

Totals use the same billing rules as exports and the billing summary page.

## Team live

1. Go to **Team** (`/team`).
2. See which members currently have a **running timer**, with project/task and elapsed time.
3. The page may update live (server-sent events) or refresh periodically.

Use this for standups or to confirm someone forgot to stop a timer.
