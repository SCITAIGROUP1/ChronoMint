# Admin: Getting started

Use the **admin app** at http://localhost:3002 (or your deployed admin URL).

## Sign in

1. Open the admin **Login** page.
2. Sign in with a workspace **ADMIN** account (demo: `admin@chronomint.dev` / `password123`).
3. Member-only accounts are meant for the client app, not admin.

## Main areas

| Page      | Path         | Purpose                            |
| --------- | ------------ | ---------------------------------- |
| Dashboard | `/dashboard` | Hours and revenue trends           |
| Projects  | `/projects`  | Create projects and invite members |
| Team live | `/team`      | Who is timing right now            |
| Billing   | `/billing`   | Hourly rates and summary           |
| Exports   | `/exports`   | Download reports                   |
| Workspace | `/workspace` | Members list                       |

## Typical first-day setup

1. **Projects** — create a project for each client or work stream.
2. **Invite links** — generate a team invite per project and send to members.
3. **Billing** — set project or user hourly rates if you track amounts.
4. **Dashboard** — confirm members are logging time after a few days.
