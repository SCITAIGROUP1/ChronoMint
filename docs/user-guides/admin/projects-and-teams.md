# Admin: Projects and teams

## Create a project

1. Go to **Projects** (`/projects`).
2. Click **Create project** (or equivalent).
3. Enter name, optional client name, color, and optional budget hours.
4. Save — an empty **team** is created for the project automatically.

## Invite members to a project

1. Open the project on the **Projects** page.
2. Open the **Team** section.
3. Click **Create invite link**.
4. Copy the link and send it to the member.
5. The member opens the link in the **client app**, signs in, and accepts.

Members must already belong to your workspace (registered or invited at workspace level).

## Manage team members

- View current team members on the project team panel.
- Deactivate or remove members when someone should no longer log time on that project (controls vary by UI version).

## What members see

Members only see projects where they are on the team. They log time against **tasks** under those projects.
